PROJECT OVERVIEW
----------------
A full-stack web application enabling students to apply for caution money refunds
after course completion. The backend is a RESTful API built with Node.js, Express,
and MongoDB, featuring a multi-role admin clearance workflow.

Tech Stack: Node.js, Express.js, MongoDB (Mongoose), JWT, BCrypt, Express-Validator
Architecture: MVC + Repository + Service pattern (modular src/ structure)
